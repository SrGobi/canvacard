# canvacard Assets

This branch holds all the assets required by canvacard.

# How to add assets?

* Clone this branch: `git clone https://github.com/SrGobi/canvacard.git -b assets`
* Add fonts/images to `data` dir
* Commit changes
* Make a pull request
